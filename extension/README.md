# OpenLongPDF Assist Chrome Extension

This unpacked Chrome extension runs on `https://chatgpt.com/*` and connects an already logged-in ChatGPT tab to a local OpenLongPDF GUI server.

Install once:

1. Open `chrome://extensions`.
2. Enable Developer mode.
3. Choose **Load unpacked**.
4. Select this `extension/` directory.

Use:

1. Start OpenLongPDF locally.
2. Click **Start Translation In ChatGPT** in the OpenLongPDF GUI.
3. Confirm the visible OpenLongPDF panel on ChatGPT.

The extension does not store ChatGPT credentials, cookies, or access tokens.
